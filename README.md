## Rodar Localmente

Clone o projeto

```bash
  git clone https://github.com/allissonneo/Final-Challenge-Gobeyond
```

Ache o Projeto Localmente

```bash
  cd Final-Challenge-Gobeyond
```

Instale as dependencias

```bash
  yarn
```

Rode o Projeto

```bash
  yarn start
```
